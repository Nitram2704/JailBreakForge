<task type="auto">
  <name>Task 2: Create config.py — Secure configuration loading</name>
  <files>src/jailbreakforge/config.py</files>
  <action>
Create `src/jailbreakforge/config.py`:

**AppConfig dataclass:**
- `provider: str` — "groq" or "openrouter" (default: "groq")
- `groq_api_key: Optional[str]`
- `openrouter_api_key: Optional[str]`
- `attacker_model: str` — model for attack generation (default: "llama-3.1-8b-instant")
- `judge_model: str` — model for verdict evaluation (default: "llama-3.1-8b-instant")
- `max_mutations: int` — max mutations per seed (default: 3)
- `target_endpoint: Optional[str]` — target API URL
- `target_model: Optional[str]` — target model name
- `target_type: str` — "openai", "anthropic", or "generic" (default: "openai")
- `categories: Optional[list[str]]` — filter attack categories (None = all)
- `output_dir: str` — report output directory (default: "reports")

**load_config() function:**
1. Call `load_dotenv()` (no-op if .env absent)
2. Read all `JAILBREAKFORGE_*` env vars
3. Validate: at least one API key must be present for the selected provider
4. If missing: print styled error to stderr with variable names and how to set them, then `raise typer.Exit(code=1)`
5. Return `AppConfig` instance
6. NEVER print or log API key values

**get_active_api_key(config: AppConfig) → str:**
Returns the API key for the active provider. Raises clean error if not set.
  </action>
  <verify>
- `uv run python -c "from jailbreakforge.config import load_config, AppConfig; print('OK')"`
- Without any API key env vars: `load_config()` exits code 1 with styled error
- With `JAILBREAKFORGE_GROQ_API_KEY=test`: `load_config()` returns AppConfig with provider="groq"
  </verify>
  <done>config.py loads API keys securely with clean error on miss, AppConfig contains all pipeline settings</done>
</task>

**models.py — Core data contracts:**

```python
@dataclass
class Attack:
    category: str          # e.g. "roleplay", "encoding"
    seed_name: str         # seed template identifier
    seed_prompt: str       # original seed prompt
    final_prompt: str      # prompt actually sent (after mutation)
    mutation_count: int    # 0 = original, 1-3 = mutated
    target_description: str # what target is being attacked

@dataclass
class JudgeVerdict:
    verdict: str           # "BLOCKED", "VULNERABLE", "PARTIAL"
    severity: str          # "CRITICAL", "HIGH", "MEDIUM", "LOW"
    reasoning: str         # why this verdict
    confidence: float      # 0.0 - 1.0
    raw_response: str      # raw judge output for debugging

@dataclass
class AttackResult:
    attack: Attack
    target_response: str
    verdict: JudgeVerdict
    timestamp: str         # ISO 8601

@dataclass
class RunConfig:
    run_id: str            # UUID
    target_name: str
    target_endpoint: str
    target_model: str
    categories: list[str]
    max_mutations: int
    provider: str
    attacker_model: str
    judge_model: str
    started_at: str        # ISO 8601

@dataclass
class RunSummary:
    config: RunConfig
    results: list[AttackResult]
    total_attacks: int
    total_blocked: int
    total_vulnerable: int
    total_partial: int
    security_score: float  # 0-100
    completed_at: str      # ISO 8601
```

These dataclasses are the contracts between ALL phases. Do not add methods beyond `__post_init__` if needed for validation.
